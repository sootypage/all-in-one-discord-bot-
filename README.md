# NovaBot

NovaBot is a full Discord bot with a web dashboard, moderation, economy, leveling, tickets, temp voice channels, alerts, reaction roles, giveaways, and music support.

## Features

### Moderation
- Ban, kick, timeout, untimeout, warn, warnings
- Clear messages
- Lock and unlock channels
- Slowmode
- Nickname and role commands
- Unban
- Mod logs
- Auto moderation
  - Anti-link
  - Anti-invite
  - Anti-spam
  - Anti-badwords

### Fun
- Ping
- Coinflip
- 8ball
- Dice
- RPS
- Joke
- Meme
- Choose
- Rate
- Trivia
- Compliment

### Economy
- Balance
- Daily
- Weekly
- Work
- Crime
- Rob
- Pay
- Deposit
- Withdraw
- Shop
- Buy
- Inventory
- Richest
- Give coins
- Add shop item
- Economy config

### Leveling
- Rank
- Leaderboard
- Profile
- Rep
- Add XP
- Level config

### Server Systems
- Tickets with categories
- Temp voice channels
- Reaction roles
- Giveaways
- Per-server config
- Dashboard controls

### Alerts
- YouTube alerts
- Twitch alerts
- GitHub alerts

### Music
- Play
- Skip
- Stop
- Pause
- Resume
- Queue
- Now Playing
- Volume

## Requirements

- Node.js 22+
- PostgreSQL database
- Discord bot token
- Discord application client secret for dashboard login
- FFmpeg
- yt-dlp
- One Opus package:
  - `@discordjs/opus`
  - or `opusscript`
